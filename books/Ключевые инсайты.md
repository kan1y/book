# Ключевые инсайты из книг других авторов

> Справочник идей и цитат для опоры при написании глав.
> Группировка по темам — чтобы при работе над конкретной главой сразу видеть, на что можно сослаться.
>
> **Правила:** пересказываем своими словами, атрибутируем явно, не выдаём за опыт автора.
> Подробно — `.claude/skills/book-coauthor/references/citing-authors.md`.

---

## Как вести этот файл

Каждая запись — по шаблону:

```
### Тема / ключевая мысль
- **Суть:** пересказ идеи своими словами (не копипаст!)
- **Источник:** автор, «Название книги»
- **Где пригодится:** глава / подглава, где эта идея уместна
- **Как использовать:** согласиться и развить / поспорить / сослаться как «копать глубже»
- **Заметки автора:** (опционально) личная реакция, связь с собственным опытом
```

Группируй записи по тематическим блокам.
Внутри блока — от более фундаментальных идей к частным.
Файлы лежат в `books/инсайты/`.

- **Гл. 11** (API и интеграции) — отдельный раздел
