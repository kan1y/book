# Глава 11. Интеграции — как системы разговаривают

_Часть II. Ядро ремесла — hard skills_

Масштабная глава для человека без бэкенд-опыта: от «что такое интеграция» до умения описать её в документе. Сквозная идея — аналитику не нужно программировать интеграции, нужно понимать принципы и однозначно их описывать.

- [x] 11.7 Форматы данных — JSON и XML — примеры, сравнение, где какой
